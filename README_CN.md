### 兼容系统 
- Linux 
- Mac OS

### 描述 
[[English](./README.md)] 从[科学](https://www.sciencemag.org/)网站下载所有播客音频和对应的文本(PDF格式)，所有的文件会存储在Science/文件夹中。

### 使用 
	$ python NatureSpider.py

注意:需要python3，[Latex](https://blog.csdn.net/Shieber/article/details/88751683)，pandoc。
